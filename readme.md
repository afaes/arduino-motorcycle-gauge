# Gauges for motorcycle


### Functions

* Speedometer
* tripmeter
* Tachometer
* Fuelmeter
* 0-60 timer :)

### Software needed
* Arduino IDE
* Stm32duino

### Parts needed

* 1 hall sensor
* 2 x 4n35 optocouplers
* 1 555-timer
* 50 ohm resistor
* 6 x 1000 ohm resistor
* 2 x 10k ohm resistor
* stm32f1c8 board
* 1 2n222 diode or similar
* 128x64 display
* 30x30cm acrylic plastic
* some cables for wiring
