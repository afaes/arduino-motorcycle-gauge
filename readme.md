#Gauges for motorcycle


###Functions

* Speedometer
* Tachometer
* Fuelmeter
* 0-60 timer
*

###Parts needed

* 1 hall sensor
* 2 4n35 optocouplers
* 1 555-timer
* 50 ohm resistor
* 1000 ohm resistor
* 10k ohm resistor
* stm32f1c8 or any arduino board
* 1 diode
* 128x64 display
* 30x30cm acrylic plastic
